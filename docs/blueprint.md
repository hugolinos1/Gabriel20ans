# **App Name**: Gabriel's Gala RSVP

## Core Features:

- Event Invitation Landing Page: A public-facing page displaying Gabriel's 20th birthday celebration details on July 18th, prominently featuring the provided image.
- Email Capture Gateway: A form on the landing page that collects the guest's email address to identify them before proceeding to the RSVP questions.
- RSVP Questionnaire Page: A dynamic form page presenting customizable questions for guests to confirm attendance, display preferences, etc., with the event image adapted as a header banner.
- Secure RSVP Submission & Storage: Handles the submission of guest responses and securely stores the attendance details and emails provided by the guests.
- Confirmation Message Display: A page shown to the guest after successful RSVP submission, confirming their participation.
- Shareable Invitation Link Generator: Creates a unique, direct URL for the RSVP form that can be embedded into an email button for easy guest access.
- RSVP Question Suggester Tool: Utilizes AI to suggest relevant and common questions for an event RSVP form, helping the event organizer quickly populate their questionnaire.

## Style Guidelines:

- Primary color: A muted, slate blue (#4A6572) inspired by the button in the example email, providing a sense of calm and reliability.
- Background color: A light, airy off-white (#F2F4F5) similar to the email's background, offering a clean and unobtrusive backdrop.
- Accent color: A subtle, warm grey (#7B8A8B) for secondary text and UI elements, complementing the blue and maintaining a sophisticated tone.
- Headline font: 'Roboto Slab' (serif) for a modern, readable, and slightly formal feel appropriate for an invitation.
- Body text font: 'Roboto' (sans-serif) for clean, easy-to-read body text that complements the headline font.
- Utilize simple, geometric icons in a line style, colored in the accent grey, for form actions and navigation to ensure a clean and modern aesthetic.
- Emphasize a clean, centrally aligned, and mobile-responsive layout, allowing the event image and form elements to take prominence without clutter. Progressive revelation of form fields will improve user experience.
- Implement subtle, smooth transitions between different steps of the RSVP form to guide the user seamlessly through the process.
